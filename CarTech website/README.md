# CarTech — Precision Car Detailing Website

A modern, responsive multi-page website for **CarTech**, a premium car detailing and ceramic coating service. Built with a sleek dark-themed UI focused on showcasing transformation, service packages, and an easy booking flow.

## Live Pages

| Page | File | Description |
|------|------|--------------|
| Home | `index.html` | Landing page introducing CarTech's premium detailing experience |
| Services | `services.html` | Essential, Signature, and Elite detailing packages |
| Portfolio | `portfolio.html` | Showroom-quality before/after gallery |
| Booking | `booking.html` | Appointment booking form for customers |

## Tech Stack

- **HTML5** — semantic page structure
- **Tailwind CSS** (via CDN) — utility-first styling with a custom dark theme
- **Google Fonts** — Inter, Outfit, and Geist for typography
- **Material Symbols** — icon set
- **Vanilla JavaScript** — interactivity and form handling

## Features

- Fully responsive layout (mobile, tablet, desktop)
- Custom dark color palette and typography scale via Tailwind config
- Dedicated service tier breakdown (Essential / Signature / Elite)
- Portfolio showcase section
- Appointment booking page with form validation
